## 追加した機能（青色ボタン）

以下のボタンは、通常の数値入力とは区別するため、**青色**で表示されています。

### 三角関数（角度制）

- `sin`：正弦関数  
- `cos`：余弦関数  
- `tan`：正接関数  

入力された数値は **度（degree）** として扱い、内部でラジアンに変換して計算しています。

---

### 対数関数

- `log`：常用対数（底 10）  
- `ln`：自然対数  

※ 入力値が 0 以下の場合は計算できないため、`Error` を表示します。

---

### 累乗計算

- `^`：べき乗演算  

例：  
`2 ^ 3 = 8`
